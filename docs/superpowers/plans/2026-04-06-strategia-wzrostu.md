# Strategia wzrostu ETFkalkulator.pl — Plan wdrożenia

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Doprowadzić ETFkalkulator.pl do 1000+ sesji organicznych miesięcznie i uruchomić pasywną monetyzację afiliacją.

**Architecture:** Dystrybucja jako marka (nie personal brand) przez grupy Facebook + Wykop + SEO. Monetyzacja przez programy afiliacyjne brokerów (XTB, Freedom24, Finax) wchodzi dopiero po osiągnięciu progu ruchu.

**Tech Stack:** Facebook Pages, Wykop, Google Search Console, programy partnerskie brokerów

---

### Task 4: Google Search Console

**Cel:** Monitorować widoczność w Google i naprawiać błędy SEO zanim zablokują wzrost.

- [ ] **Krok 1: Dodaj właściwość**

  Wejdź na: search.google.com/search-console  
  Zaloguj się kontem Google (najlepiej tym samym co GA4)  
  Kliknij "Dodaj właściwość" → wybierz "Domena"  
  Wpisz: `etfkalkulator.pl`

- [ ] **Krok 2: Zweryfikuj domenę**

  Metoda: rekord DNS TXT (wymaga dostępu do panelu domeny u rejestratora)  
  Skopiuj token weryfikacyjny z Search Console  
  W panelu domeny dodaj rekord TXT z tym tokenem  
  Wróć do Search Console → kliknij "Zweryfikuj"  
  (Propagacja DNS: 1–24h)

- [ ] **Krok 3: Prześlij sitemap**

  Po weryfikacji: Sitemaps → dodaj URL:  
  `https://etfkalkulator.pl/sitemap.xml`  
  Kliknij "Prześlij"

- [ ] **Krok 4: Sprawdź stan indeksowania**

  Po 48h: Przejdź do "Strony" → sprawdź czy strony są zaindeksowane  
  Jeśli strony są w "Odkryte — aktualnie nieindeksowane" → kliknij URL → "Poproś o indeksowanie"  
  Priorytet: kalkulator-etf.html, kalkulator-obligacji.html, blog/index.html

- [ ] **Krok 5: Ustaw cotygodniowy przegląd**

  Każdy poniedziałek: 15 minut w Search Console  
  Sprawdzaj: nowe słowa kluczowe, błędy indeksowania, Core Web Vitals
